## Installation

Clone the repository:

```bash
git clone https://github.com/LevoKMH/portfolio.git
```

Open the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

---

## Build

Production build:

```bash
npm run build
```

Preview build:

```bash
npm run preview
```
